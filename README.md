## 英雄对战DAPP


### 一、选题背景，依据

来源是手机游戏上的一类挂机游戏，相比于传统的移动端游戏，挂机类的游戏有以下特点

- 所需要的操作更少。
- 动画，物理引擎依赖程度低
- 大部分由文字组成

综上，这类游戏属于比较有趣，玩起来时间比较少，适合现在较快的生活节奏，所以这类游戏在移动游戏市场越发受欢迎。
